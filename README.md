# Flashcards - TIEP111 Harjoitustyö

Simple GUI Flashcard application written in Java with the JavaFX application framework.

## Features  

### Deck View  
![Deck View UI Screenshot](/data/deck_view.png)  
- View through created decks  
- Create new decks  
- Delete unwanted decks  
- Access Flashcard View by double-clicking on a deck  
- Access Edit View  

### Flashcard View  
![Flashcard View UI Screenshot](/data/card_view.png)  
- Go through the cards in a deck in a random order  
- Flip cards to see their description  

### Edit View
![Edit View UI Screenshot](/data/edit_view.png)  
- Edit deck title and description
- Create new cards
- Click on the card list to select a card
- Edit the title and description of the selected card
- Delete the selected card
